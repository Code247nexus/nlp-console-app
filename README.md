# nlp-console-app
A Python-based NLP CLI application offering chatbot, code generation, summarization, translation, and grammar correction using NLP Cloud API.


NLP CLI Application

This project is a console-based Natural Language Processing application built using Python and the NLP Cloud API.

Features:
- User registration and login system  
- AI-powered chatbot interaction  
- Automatic code generation from text prompts  
- Text summarization  
- Language translation  
- Grammar and spelling correction  

Technologies Used:
- Python  
- Object-Oriented Programming  
- NLP Cloud API  
- Command Line Interface  

The application provides an interactive menu-driven interface that allows users to access multiple NLP services in a single program.

This project demonstrates:
- API integration  
- User authentication logic  
- Modular programming  
- Real-world NLP use cases  

Future Improvements:
- Add GUI interface  
- Secure authentication system  
- Support for more languages  
- Better error handling
